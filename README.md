# Traffic Light – setInterval Practice

A simple JavaScript project that simulates a traffic light system.  

---

## Features
- Automatic light cycle: **Red → Yellow → Green → Red**
- Timing controlled with **setInterval()**
- Clean and simple UI using HTML & CSS
- Exercise for understanding JavaScript timers

---

## Technologies Used
- **HTML5**
- **CSS3**
_ **React**
- **JavaScript (ES6+)**
